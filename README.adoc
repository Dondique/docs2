=== Алгоритм согласования в виде списка

. *Создание черновика*
.. Создать файл в формате AsciiDoc
.. Заполнить титульную информацию

. *Коммит и Pull Request*
.. Закоммитить изменения с описанием
.. Создать PR с ссылкой на задачу в Jira

. *Code Review*
.. Назначить минимум двух ревьюверов
.. Ответить на комментарии

. *Слияние*
.. После одобрения выполнить squash-merge
.. Удалить ветку

. *Публикация*
.. Автоматический деплой через GitHub Actions
.. Уведомление в чат команды

== Дополнительные возможности AsciiDoc

=== Вложенные таблицы
В AsciiDoc можно создавать таблицы внутри таблиц [citation:5][citation:7]. Для этого внутренняя таблица использует другие разделители:
[cols="1,2a", options="header"]
|===
| Категория
| Детали

| Фрукты

[cols="1,1", options="header"]
!===
! Название
! Цвет

! Яблоко
! Красное

! Банан
! Жёлтый
!===

| Овощи

[cols="1,1", options="header"]
!===
! Название
! Цвет

! Морковь
! Оранжевый

! Брокколи
! Зелёный
!===
|===

[NOTE]
====
Вложенные таблицы — мощный инструмент, но их стоит использовать умеренно, так как они могут усложнить восприятие документа [citation:5][citation:7].
====
