## 📋 Описание проекта

Этот проект создан для тестирования подхода **Docs as Code** — когда документация хранится в Git-репозитории рядом с кодом.

### Цели тестирования:
- Проверить работу Git на локальном ПК
- Научиться создавать, редактировать и синхронизировать документы
- Оценить удобство работы с Markdown-разметкой
